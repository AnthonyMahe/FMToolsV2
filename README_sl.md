> Orodja za ocenjevanje za Football Manager 26

Dobrodošli v orodju za ocenjevanje igralcev in osebja

## Description
Po številnih testih v Football Managerju v zadnjih letih sta nam Kinito in DoctorDim omogočila poglobljeno razumevanje pomembnih vidikov igre.

## Features
- **Skriti atributi**: Ocenite skrite atribute igralca z uporabo: Osebnost, Ravnanje z mediji, Odločnost
- **Osebje**: Ocenite osebje za najbolje zadovoljevanje vaših potreb. V FM26 so atributi prikazani z opisnimi besedami.
- **Poročila**: Interpretirajte različna poročila: Skavt, Trener, Individualni trening
- **Infrastruktura**: Spoznajte in ocenite razvoj infrastrukture vašega kluba.
