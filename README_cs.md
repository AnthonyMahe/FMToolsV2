> Hodnotící nástroje pro Football Manager 26

Vítejte v nástroji pro hodnocení hráčů a realizačního týmu

## Description
Po četných testech ve Football Manager v posledních letech nám Kinito a DoctorDim umožnili hluboce pochopit důležité aspekty hry.

## Features
- **Skryté atributy**: Vyhodnoťte skryté atributy hráče pomocí: Osobnosti, Práce s médii, Odhodlání
- **Realizační tým**: Vyhodnoťte realizační tým pro nejlepší splnění vašich potřeb. V FM26 jsou atributy zobrazeny popisnými slovy.
- **Zprávy**: Interpretujte různé zprávy: Skaut, Trenér, Individuální trénink
- **Infrastruktura**: Poznejte a vyhodnoťte rozvoj infrastruktury vašeho klubu.
