> Novērtēšanas rīki Football Manager 26

Laipni lūdzam spēlētāju un personāla novērtēšanas rīkā

## Description
Pēc daudziem Football Manager testiem pēdējos gados, Kinito un DoctorDim ļāva mums dziļi izprast svarīgus spēles aspektus.

## Features
- **Slēptie atribūti**: Novērtējiet spēlētāja slēptos atribūtus, izmantojot: Personību, Mediju apstrādi, Apņēmību
- **Personāls**: Novērtējiet personālu, lai vislabāk apmierinātu jūsu vajadzības. FM26 atribūti tiek parādīti ar aprakstošiem vārdiem.
- **Ziņojumi**: Interpretējiet dažādus ziņojumus: Skauts, Treneris, Individuālais treniņš
- **Infrastruktūra**: Iepazīstiet un novērtējiet sava kluba infrastruktūras attīstību.
