> Bewertungstools für Football Manager 26

Willkommen beim Spieler- und Trainerstab-Bewertungstool

## Description
Nach zahlreichen Tests am Football Manager in den letzten Jahren haben Kinito und DoctorDim uns ermöglicht, wichtige Aspekte des Spiels tiefgreifend zu verstehen.

## Features
- **Versteckte Attribute**: Bewerten Sie die versteckten Attribute eines Spielers anhand von: Persönlichkeit, Medienumgang, Entschlossenheit
- **Trainerstab**: Bewerten Sie den Stab, um Ihren Bedürfnissen optimal zu entsprechen. In FM26 werden Attribute mit beschreibenden Wörtern angezeigt.
- **Berichte**: Interpretieren Sie verschiedene Berichte: Scout, Trainer, Einzeltraining
- **Infrastruktur**: Kennen und bewerten Sie die Infrastrukturentwicklung Ihres Vereins.
