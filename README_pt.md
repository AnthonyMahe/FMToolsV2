> Ferramentas de avaliação para Football Manager 26

Bem-vindo à ferramenta de avaliação de jogadores e equipa técnica

## Description
Após numerosos testes no Football Manager nos últimos anos, Kinito e DoctorDim permitiram-nos compreender aspetos importantes do jogo em profundidade.

## Features
- **Atributos ocultos**: Avalia os atributos ocultos de um jogador usando: Personalidade, Gestão de media, Determinação
- **Equipa técnica**: Avalia a equipa técnica para satisfazer melhor as tuas necessidades. No FM26, os atributos são mostrados com palavras descritivas.
- **Relatórios**: Interpreta diferentes relatórios: Observador, Treinador, Treino individual
- **Infraestrutura**: Conhece e avalia o desenvolvimento da infraestrutura do teu clube.
