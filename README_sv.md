> Utvärderingsverktyg för Football Manager 26

Välkommen till utvärderin gsverktyg för spelare och personal

## Description
Efter många tester i Football Manager under de senaste åren har Kinito och DoctorDim gett oss möjlighet att djupt förstå viktiga aspekter av spelet.

## Features
- **Dolda attribut**: Utvärdera en spelares dolda attribut med: Personlighet, Mediahantering, Beslutsamhet
- **Personal**: Utvärdera personalen för att bäst möta dina behov. I FM26 visas attribut med beskrivande ord.
- **Rapporter**: Tolka olika rapporter: Scout, Tränare, Individuell träning
- **Infrastruktur**: Känn till och utvärdera din klubbs infrastrukturutveckling.
