Tere tulemast mängijate ja personali hindamise tööriista juurde

## Description
Pärast arvukaid Football Manageri teste viimastel aastatel on Kinito ja DoctorDim võimaldanud meil mängu olulisi aspekte sügavuti mõista.

## Features
- **Peidetud omadused**: Hinnake mängija peidetud omadusi kasutades: Isiksus, Meediakäsitlemine, Otsustavus
- **Personal**: Hinnake personali, et paremini vastata teie vajadustele. FM26-s kuvatakse omadusi kirjeldavate sõnadega.
- **Aruanded**: Tõlgendage erinevaid aruandeid: Skaudi, Treeneri, Individuaalne treening
- **Infrastruktuur**: Tundke ja hinnake oma klubi infrastruktuuri arengut.
