> Narzędzia oceny dla Football Manager 26

Witamy w narzędziu do oceny zawodników i sztabu

## Description
Po licznych testach w Football Manager w ostatnich latach, Kinito i DoctorDim pozwolili nam dogłębnie zrozumieć ważne aspekty gry.

## Features
- **Ukryte atrybuty**: Oceń ukryte atrybuty gracza używając: Osobowości, Obsługi mediów, Determinacji
- **Sztab**: Oceń sztab, aby najlepiej spełnić twoje potrzeby. W FM26 atrybuty są wyświetlane za pomocą słów opisowych.
- **Raporty**: Interpretuj różne raporty: Skaut, Trener, Trening indywidualny
- **Infrastruktura**: Poznaj i oceń rozwój infrastruktury swojego klubu.
