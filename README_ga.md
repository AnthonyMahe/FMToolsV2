> Uirlisí meastóireachta do Football Manager 26

Fáilte go dtí an uirlis meastóireachta d'imreoirí agus d'fhoireann

## Description
Tar éis go leor tástálacha ar Football Manager le blianta beaga anuas, chuir Kinito agus DoctorDim ar ár gcumas gnéithe tábhachtacha den chluiche a thuiscint go domhain.

## Features
- **Tréithe folaithe**: Meastóireacht a dhéanamh ar thréithe folaithe imreora ag úsáid: Pearsantacht, Láimhseáil meán, Diongbháilteacht
- **Foireann**: Déan meastóireacht ar an bhfoireann chun do riachtanais a shásamh. In FM26, taispeántar tréithe le focail thuairisciúla.
- **Tuairiscí**: Léirmhínigh tuairiscí éagsúla: Scout, Traenálaí, Traenáil aonair
- **Bonneagar**: Aithin agus meastóireacht a dhéanamh ar fhorbairt bonneagair do chlub.
