> Hodnotiace nástroje pre Football Manager 26

Vitajte v nástroji na hodnotenie hráčov a realizačného tímu

## Description
Po početných testoch vo Football Manager v posledných rokoch nám Kinito a DoctorDim umožnili hlboko pochopiť dôležité aspekty hry.

## Features
- **Skryté atribúty**: Vyhodnoťte skryté atribúty hráča pomocou: Osobnosť, Práca s médiami, Odhodlanie
- **Realizačný tím**: Vyhodnoťte realizačný tím pre najlepšie splnenie vašich potrieb. V FM26 sú atribúty zobrazené opisnými slovami.
- **Správy**: Interpretujte rôzne správy: Skaut, Tréner, Individuálny tréning
- **Infraštruktúra**: Spoznajte a vyhodnoťte rozvoj infraštruktúry vášho klubu.
