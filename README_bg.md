> Инструменти за оценка за Football Manager 26

Добре дошли в инструмента за оценка на играчи и персонал

## Description
След множество тестове във Football Manager през последните години, Kinito и DoctorDim ни позволиха да разберем в дълбочина важни аспекти на играта.

## Features
- **Скрити атрибути**: Оценете скритите атрибути на играч чрез: Личност, Работа с медии, Решителност
- **Персонал**: Оценете персонала за най-добро задоволяване на вашите нужди. Във FM26 атрибутите се показват с описателни думи.
- **Доклади**: Интерпретирайте различни доклади: Скаут, Треньор, Индивидуална тренировка
- **Инфраструктура**: Опознайте и оценете развитието на инфраструктурата на вашия клуб.
