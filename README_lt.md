> Vertinimo įrankiai Football Manager 26

Sveiki atvykę į žaidėjų ir personalo vertinimo įrankį

## Description
Po daugybės Football Manager testų pastaraisiais metais, Kinito ir DoctorDim leido mums giliai suprasti svarbius žaidimo aspektus.

## Features
- **Paslėpti atributai**: Įvertinkite žaidėjo paslėptus atributus naudodami: Asmenybė, Žiniasklaidos valdymas, Ryžtas
- **Personalas**: Įvertinkite personalą, kad geriausiai atitiktų jūsų poreikius. FM26 atributai rodomi aprašomaisiais žodžiais.
- **Ataskaitos**: Interpretuokite įvairias ataskaitas: Skautai, Treneriai, Individualios treniruotės
- **Infrastruktūra**: Pažinkite ir įvertinkite savo klubo infrastruktūros plėtrą.
