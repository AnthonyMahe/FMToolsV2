> Vurderingsværktøjer til Football Manager 26

Velkommen til værktøjet til vurdering af spillere og personale

## Description
Efter talrige tests i Football Manager de seneste år har Kinito og DoctorDim gjort det muligt for os at forstå vigtige aspekter af spillet i dybden.

## Features
- **Skjulte egenskaber**: Vurder en spillers skjulte egenskaber ved hjælp af: Personlighed, Mediehåndtering, Beslutsomhed
- **Personale**: Vurder personale for bedst at opfylde dine behov. I FM26 vises egenskaber med beskrivende ord.
- **Rapporter**: Fortolk forskellige rapporter: Scout, Træner, Individuel træning
- **Infrastruktur**: Kend og vurder din klubs infrastrukturudvikling.
