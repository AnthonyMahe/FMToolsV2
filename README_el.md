## Description
Μετά από πολυάριθμες δοκιμές στο Football Manager τα τελευταία χρόνια, ο Kinito και ο DoctorDim μας επέτρεψαν να κατανοήσουμε σε βάθος σημαντικές πτυχές του παιχνιδιού.

## Features
- **Κρυφά χαρακτηριστικά**: Αξιολογήστε τα κρυφά χαρακτηριστικά ενός παίκτη χρησιμοποιώντας: Προσωπικότητα, Διαχείριση Μέσων, Αποφασιστικότητα
- **Προσωπικό**: Αξιολογήστε το προσωπικό για να καλύψετε καλύτερα τις ανάγκες σας. Στο FM26, τα χαρακτηριστικά εμφανίζονται με περιγραφικές λέξεις.
- **Αναφορές**: Ερμηνεύστε διάφορες αναφορές: Ανιχνευτής, Προπονητής, Ατομική προπόνηση
- **Υποδομές**: Γνωρίστε και αξιολογήστε την ανάπτυξη υποδομών του συλλόγου σας.
